OUR PROGRAMS


Special Camp(IFYP)

ပထမဆုံးတစ်ခု Special Camp ကတော့ ကျွန်တော်တို့ လျှောက်တဲ့သူတွေ အယောက် အများကြီးထဲကမှ တကယ် Passion ဖြစ်တဲ့ လူ 25 ယောက်ကိုသာ ခေါ်ယူပါတယ်။ သူက စနေ တနင်္ဂနွေပဲ တက်ရပေမဲ့ မနက် 9 နာရီကနေ ညနေ 3ခွဲအထိ ရှယ်သင်ရပါတယ်။ သင်တန်းကာလက နှစ်လကြာတယ်။သင်တဲ့ဘာသာတွေက Web Development နဲ့ Programmingပေါ့။...


Normal Program

ဒါကတော့ အရှေ့ကလို အားကြီးနဲ့ တစ်နေကုန်မသင်ပါဘူး။ Python တစ်ခုထဲကိုပဲ တစ်နေ့နှစ်နာရီနဲ့ရှယ်သင်ပေးမှာပါ။ ဒီအတန်းက များသောအားဖြင့် တစ်ပတ်နှစ်ရက်ပဲ တစ်နေ့နှစ်နာရီသင်တယ်ဆိုပေမဲ့ ကျောင်းသားတွေမှာ Zero Knowledgeတွေပါနေရင်တော့ တစ်ပတ်ကို 3 ရက်လောက်ခေါ်ပါတယ်။...


Paid Program

Paid Program ဆိုတော့ ပိုက်ဆံပေးတက်ရမှာပေါ့ဟုတ်လားလို့ မေးရင် ဟုတ်ပါတယ်။ ဒါပေမဲ့ ဘယ်လောက်ပေးပါ အများကြီးပေးပါ မတောင်းထားတဲ့ Paid Program ပါ။ သင်တန်းကြေးလားဆိုတော့မဟုတ်ပါဘူး Donationတောင်းတာပါ။ ကိုယ်ကြိုက်သလောက် နည်းတယ်များတယ် မသတ်မှတ်ပဲ..
